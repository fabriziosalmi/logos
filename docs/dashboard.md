# Dashboard

The dashboard is served at:

```
GET /
```

It provides:

- Live preview for the current parameter set
- Copyable render URL
- Animation grid with “copy URL” interaction
- Icon pack browser (Lucide and Heroicons) with search and selection

## Offline deployment

The dashboard assets (HTML/CSS/JS) are served locally from the same origin as the API. No external CDNs are required at runtime.
